How to use this editor

  1. Edit any project — click a card to expand it, change text fields directly.
  2. To add images: put your image files in assets/projects/[slug]/ then type the path (e.g. assets/projects/tact-monster/preview.jpg).
  3. When done, click Download projects.json — this saves the file to your Downloads.
  4. Replace the old projects.json in your project folder with the downloaded one.
  5. Push to GitHub: git add . && git commit -m "update" && git push
